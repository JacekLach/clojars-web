Clojars is a dead easy community repository for open source Clojure libraries.

pushing with leiningen

$ lein pom
$ scp pom.xml mylib.jar clojars@clojars.org:

It's the default repository for Leiningen but you can use it with other build tools like Maven or Ivy as well.

maven repository

<repository>
  <id>clojars.org</id>
  <url>http://clojars.org/repo</url>
</repository>

To get started pushing your own jars create an account and then check out the tutorial.
